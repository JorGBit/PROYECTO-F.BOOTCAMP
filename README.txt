# PROYECTO-F.BOOTCAMP
Este proyecto es el diseño backend y frontend de una aplicación de notas que permite hacer las siguientes funciones:

-Registrar un usuario
-Logear usuario
-Crear una nota (título, texto y categoría)
-Visualizar la información de la nota
-Visualizar un listado de notas. En la lista sólo se ven los títulos
-Modificar la información que se guarda en la base de datos con el objetivo de corregir la información de una nota y volverla a guardar
-Eliminar una nota guardada
-Asociar una imagen a una nota (a fecha de 6-3-23 esta funcionalidad está incompleta)

 

Para que este proyecto funcione debemos:

 *BACKEND*
 -Seguir los pasos que aparecen en el archivo README
 
 *FRONTEND* 
 -Instalar el archivo node.modules
 -Descargar la carpeta notes-frontend
 -Crear un archivo .env con las variables de entorno necesarias. Crear un REACT_APP_BACKEND= como variable de entorno y poner a la escucha.
